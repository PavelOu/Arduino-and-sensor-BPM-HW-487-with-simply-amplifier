with connectors RS232 or USB converters to RS232

Basic RS232 commands:
= = = = = = = = = = =
This commans are readable in program SerialPlotter version 10 and 11 .
Command	Answer
- - - - - - - -
A	*A,400,#,CRLF   number is readed analog data example
B	*B,399,#,CRLF	Cr is carriage return, LF is Line Feed
C	*,401,#,CRLF	CRLF are marks from RS232 protocol
D	398CRLF		simply data for Arduino IDE plotter for example	
E20	*P,20,400,500,100,#,CRLF => returned next parameters	
	20, data time step in mikroseconds
	400, off-line readed data with osciloscope
	500, max - " -
	100, offset for readed data (usually zero)
F	=> read all transfer detail comments from Arduino (for tests)
G	=> test of yellow LED diode installed by Arduino (optionally)
H	=> test of red RED diode installed by Arduino (optionally)
J	=> return analog input (A0 to A7) 
          and if data are normal (0) or invered (1)
I	*I,nr,#, CRLF => test of aktual connected device nr
K1	*K,1,#,CRLF example => 1 is slave number in the ESP-NOW master-slaves system
(K1	*K,0,#,CRLF example => zero answer inform that this slave number 1 is off)
L300	*P,20,300,500,100,#,CRLF => off-line readed data with osciloscope
M	*M,20,300,6000,100,#,CRLF => off-line read data to internal memory, 6000 is estimated time	
N	*N,20,300,6000,100,300 times of data on,#,CRLF = read 300 data from internal slave device memory
O10	*P,20,300,500,10,#,CRLF => change in offset for data
P	*P,20,300,500,10,#,CRLF => read parameters
Q	= stop of reading details after commands
R	*R,20,300,time,10,#,CRLF => is time of previous data measuring in microseconds
S	*S,20,300,time,10,#,CRLF => stop of reading B,C,D data in cycle
T	*T,dataWaitTime,#,CRLF => wait time for ESP
U	*U,InverseDat,#,CRLF => 0 = data normal, 1 = data are in device inverted
V	*V,dataPin,#,CRLF => analog input pin can be 0 to 7

Note:
After commands B,C and D are data sended in cycle till S command is received

All this commands can product Arduino with sensores
with my sketch BPM_senzor_HW_487_en_5.ino

These commands are the same for my ESP-NOW system Master - Quick Slaves (M-QS)
(later re-programmed in SerialOsciloscope uprade)

Simply manual how to use SerialPlotter14
= = = = = = = = = = = = = = = = = = = = =

Main(home) window
- - - - - - - - -
- after main(home) window is on and OK, push Device Set Button
- or Com Button, if you are not sure if your com port is set well
- Open Port window will be shown

Open Com port window
- - - - - -
- push Open button
- if handle of predefined com port will set on, push Purge button and purge this comm port
- choose Device Set  button and Device Set  window will be shown

Device Set window
- - - - - - - - -
- choose Q command from Call combobox an push Call button
- if answer will be OK and green trigger light will be on, you can continue with next parameters
- for example choose G Y Led from Call combobox and push Call button
- yellow led on your device (if is installed) will light on
- choose your input device from Devices combobox
- choose I command from Commands combobox
- push Set button and wait  for answer in Answer or Result field
- if your asked device is on, this one will be confirmed
- choose K command from Serial Commands combobox
- push Set button and wait  for answer in Answer or Result field
- when this answer will repeat your slave device number, this one is on-line
- if your choosed device is on, it will be confirmed, else zero will be in answer field
- push Read Par button to read pre-set parameters for Measure windows
- push Data button to read data from your device via command A to be sure your device measured OK 

Measure window
- - - - - - - -
- choose your task from listbox with tasks
- choose plotter type from plotter combobox A or B (more quick when data reading)
- I prefer and test onpz B command for plotter, A or C,D I did not use or test, no time, I am sorry
- if you need record of data, cross Record checkbox, New file button will be wisible
- occasionally write note of you to Note field
- press New file button to set the header of this new file what will be used for next data registration
- press Graph and  Graph window will be shown

Grapf Form window
- - - - - - - - - 
(Note - Grapf Form is window for direct measurig, Graph Image is for data seeing after measuring)
- set in Radiobutton Time axe for the time graph axe shift 
  after this time will be over the end of time range
- see and follow your measuring data, then press Stop button
- after all measuring ends, press Task button
- confirm if you want to save your data or on your hard disc
  or into databox onlz for the next analysation in tabs or graphs.
- press Menu button and Menu window will be shown

Menu window
- - - - - -
- clik on the column with data
- choose Tab Par or Graph Par
- set parameters for table or graph
- click on the Table or Graph image buttons
